如何搭建博客

##创建博客步骤
1.下载git 注册账号
2.下载node.js
3.使用npm安装Hexo 在指定文件夹运行cmd npm install -g hexo-cli
4.创建 hexo init 进入cd 运行 npm install 初始化后会有一个项目文件夹
5.更改_config.yml文件
6.进入项目文件夹 新建博客 hexo new “xxx” 生成 hexo g
7.hexo server 启动网站 http://localhost:4000
8.安装 hexo-deployer-git 运行 npm install hexo-deployer-git –save
9.部署 hexo deploy

具体步骤参考 hexo.com
